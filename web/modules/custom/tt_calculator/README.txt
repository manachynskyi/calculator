This module is created for calculation and saving total price, based on user age and car size.


INSTALLATION
------------

 * Install as you would normally install a Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------
The calculator is placed in block(Calculator block), so you can add it to any part of site (block layout, page manager, panels ...).

Module provide form, where you can change default values for fixed and variable price  <sitename>/admin/config/calculator-settings .

Also, you can check last 20 submissions on <sitename>/admin/config/show-submissions
